# Open Data Sources for Urban Mobility & Sustainability Assessment

This repository provides a curated list of the **web-based open data sources** used for the external evaluation (**Track 2**) of urban mobility transitions in the UP2030 project. These sources ensure transparency, reproducibility, and cross-city comparability [3-5].

### 🌐 Harmonized Open Data Catalog

| Resource Name | Description | Direct Access Link |
| :--- | :--- | :--- |
| **ITDP Atlas** | Provides indicators on population access to cycleways, frequent public transport, and essential services. | [https://atlas.itdp.org/](https://atlas.itdp.org/) |
| **MIT Overpass (OSM)** | Used for spatial queries on OpenStreetMap to compute cycling network density and transit stop distribution. | [https://overpass-turbo.eu/](https://overpass-turbo.eu/) |
| **Copernicus Data Space** | Source for Sentinel-2 and SAR satellite imagery to calculate vegetation indices (NDVI) and surface permeability. | [https://dataspace.copernicus.eu/](https://dataspace.copernicus.eu/) |
| **CAMS Atmosphere** | Provides air quality reanalysis for traffic-related pollutants such as Carbon Monoxide (CO) and Sulphur Dioxide (SO2). | [https://atmosphere.copernicus.eu/](https://atmosphere.copernicus.eu/) |
| **Global Covenant of Mayors** | Repository for city-level Greenhouse Gas (GHG) emission inventories and transport-related reports. | [https://www.globalcovenantofmayors.org/](https://www.globalcovenantofmayors.org/) |
| **OECD Data** | Harmonized socioeconomic indicators, including GDP per capita growth and regional poverty rates. | [https://www.oecd.org/en/data.html](https://www.oecd.org/en/data.html) |
| **Meteostat** | Historical weather records for temperature and precipitation used to assess urban resilience context. | [https://meteostat.net/](https://meteostat.net/) |
| **Eurostat** | Primary source for European demographic statistics and harmonized urban data. | [https://ec.europa.eu/eurostat/](https://ec.europa.eu/eurostat/) |
| **JRC - LUCAS Topsoil** | Data on physical soil properties, including water capacity and composition for resilience studies. | [https://esdac.jrc.ec.europa.eu/](https://esdac.jrc.ec.europa.eu/) |

### 🛠 Evaluation Framework
The project adopts a **dual-track KPI system**:
1. **Track 1 (Local):** City-reported KPIs focused on participatory governance and pilot-scale implementation.
2. **Track 2 (External):** Independent monitoring using the **open data sources** listed above to capture city-wide structural trends.
